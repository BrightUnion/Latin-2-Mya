## 使用方法

此Python文件可用于拉丁转录字母转换成缅甸语

请将需要转录的拉丁转录字母放在“缅语原文.txt”文件内，然后放在与此Python文件同目录文件夹下，运行此Python代码使用。

基于 Wwin_Burmese1 TTF 文件转录字符规则制作。
